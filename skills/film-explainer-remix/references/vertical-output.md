# Vertical Output

Use for Douyin/Kuaishou/TikTok/Reels-style 9:16 previews.

## Layout Choices

- blurred background + centered source: safest for widescreen films
- smart crop: use only when faces/actions remain visible
- split layout: top hook text or face crop, bottom source action
- card layout: use for maps, rules, timelines, or complex exposition

## Safe Areas

- Keep main subtitles away from platform captions, like/comment/share buttons, and source hard subtitles.
- Do not place crucial clues at the extreme right side.
- Keep protagonist face and key props near center when possible.

## First Frame

The first frame should work as a cover:
- visible face or danger object
- 4-9 Chinese characters of high-stakes text
- no tiny UI unless zoomed
- no dark unreadable frame

## Cropping Rules

- If a shot loses its meaning after vertical crop, use blurred-background layout instead.
- For UI or small props, zoom or freeze before using vertical crop.
- For action scenes, avoid random center crop; choose a segment where movement remains readable.

## Suggested Workflow

1. Build horizontal pilot to verify story.
2. Convert the strongest 45-90 seconds to vertical.
3. Adjust subtitle position and first-frame text.
4. Only then scale to full duration.
