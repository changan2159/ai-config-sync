# Benchmark Analysis

Use when the user provides reference explainer videos, asks why a piece feels weaker than others, or wants a style close to mature/breakout accounts.

## What To Extract

Analyze reference videos as systems, not as content to copy:
- opening mechanism: first frame, first sentence, first sound, first cut
- promise: what question the viewer is waiting to resolve
- cut rhythm: average shot length in the first 30 seconds and during action/reveal sections
- narration design: sentence length, voice energy, suspense wording, silence placement
- subtitle packaging: font size, keyword emphasis, line breaks, outline, safe area
- visual packaging: push-ins, black cards, red boxes, arrows, freeze frames, glitches, maps, UI zooms
- audio stack: BGM mood, hit points, source sound moments, ducking
- loop design: where each new question is opened and closed
- ending strategy: emotional close, irony, sequel tease, or thesis line

## Comparison Table

| Dimension | Reference Pattern | Current Draft | Gap | Fix |
| --- | --- | --- | --- | --- |
| Hook | crisis/contradiction/outcome | chronology/lore | weak stop-scroll | rewrite first 8s |
| Shot Density | 8-15 shots/30s | long source blocks | visually flat | split beats into proof/reaction/pressure |
| Audio | voice + BGM + hits | voice over raw source | low energy | duck source, add hits, preserve effects |
| Subtitles | short with emphasis | paragraph blocks | hard to read | split and style |
| Packaging | designed first frame | random frame | weak click/retention | build cover/first-frame |

## Rules

- Do not copy wording, title, thumbnail, or unique edit sequence from a reference.
- Do not claim live platform trends unless actual reference samples were analyzed.
- If no references are provided, state that the output uses a generic mature explainer workflow and recommend 3-5 benchmark samples for style matching.
- Treat benchmark findings as constraints for the next version: hook pattern, shot density, subtitle style, BGM energy, and visual devices.
