# Retention Map

Use before writing the script. The goal is to make each segment create a reason to keep watching.

## Core Principle

Chronology is not the structure. Curiosity is the structure.

A strong explainer usually orders information like this:
1. danger/result/contradiction
2. protagonist identity
3. immediate obstacle
4. rule or hidden cost
5. first plan
6. plan failure
7. truth/reversal
8. final choice and price
9. emotional landing

## Opening Patterns

Use one dominant pattern:
- impossible result: `他醒来时，整个基地只剩下他的呼吸声。`
- deadline: `再过四分钟，这艘船会把所有人送进火里。`
- contradiction: `她明明救了全城，却被系统判定为最高威胁。`
- taboo choice: `为了救一个孩子，他必须先关掉所有人的氧气。`
- hidden enemy: `他以为追杀自己的是怪物，直到看见监控里的自己。`

Avoid:
- year/place/worldbuilding as the first sentence
- character résumé before danger
- explaining system rules before showing their cost

## Beat Density

Every 15-25 seconds, add one turn:
- new danger
- new clue
- failed attempt
- stronger enemy
- betrayal or hidden rule
- irreversible choice
- body cost or emotional cost

If a segment only explains background, compress it into one line and cover it with the strongest visual clue.

## Protagonist-View Language

Prefer:
- `他看见...`
- `他发现...`
- `他意识到...`
- `他不敢...`
- `他选择...`
- `他付出的代价是...`

Avoid:
- omniscient encyclopedia tone
- abstract theme statements before the ending
- generic summaries with no action verb

## Mini-Structure For 45-90 Second Pilot

- 0-5s: strongest crisis or contradiction
- 5-15s: protagonist + location + immediate problem
- 15-35s: first attempt and discovered rule
- 35-60s: plan fails or truth appears
- 60-80s: choice/cost/tease
- final 5-10s: close or cliffhanger depending on user goal

## Mini-Structure For 3-5 Minute Explainer

- 0-8s: hook
- 8-30s: lock-in
- 30-60s: world rule through action
- 60-120s: mission and first failure
- 120-210s: larger truth and strongest reversal
- 210-270s: final choice and consequence
- last 10-20s: emotional/thematic closure
